Article 1.0 React

  1. Retour

Auteur: Jeremy Sfez


Quels sont les points favorables de React qui influent le choix d'une entreprise ?

Premièrement, le contenu est référençable grâce à l’utilisation d’un serveur Node. Cela permet de générer le code côté client et serveur.
React est de plus très performant grâce à son DOM virtuel qui permet à chaque rafraîchissement d’actualiser uniquement les parties impactées par les modifications.
Ce framework est basé sur des composants réutilisables, testables et combinables.
De plus, celui-ci utilise la syntaxe JSX, qui permet de décrire une structure DOM directement en JavaScript dans un langage très proche du HTML.
Cette syntaxe, fortement recommandée, permet de faciliter l’écriture et la lecture.

Enfin ce framework ne se limite pas à la création d’application web, on peut l’utiliser pour beaucoup d’autres choses :
- react-native pour la création d’application mobiles
- react-sketch pour la création de maquette sketch
- react-word pour la création de documents word
- react-pdf pour la création de documents PDF

Cela fait plusieurs bons points non négligeables qui démarquent ce framework des autres et jouent sur le choix des entreprises.
