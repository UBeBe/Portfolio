Veille Technologique

Dans le cadre du BTS, nous avions pour mission de réaliser une veille technologique.
Le sujet de ma veille est React. C'est une librairie JavaScript que j'ai pu découvrir et utiliser en stage de deuxième année.

Cette veille a été réalisée à l'aide des outils suivants :
- daily.dev, qui est un blog communautaire regroupant pleins d'informations concernant un sujet en particulier
- Google Alerte, en utilisant le mot-clé "React"
Je me suis également servi d'application de notes sur mon téléphone portable afin de regrouper les différents sites me servant de sources, mais les informations viennent essentiellement du site officiel de React : https://reactjs.org/
Vous pouvez cliquer ici pour voir mon profil daily.dev
